这是星网宇达组合导航系统（型号XWGI5651）的导航信息采集节点。
接收XWGI5651输出的GPFPD（该系统自定义的通信协议）信息，并在ROS中广播。
该节点发出的msg有两种，一种是按gpfpd格式输出，另一种按POS320的msg格式输出，具体格式见相应的msg文件。
GPFPD协议的具体定义见本目录下的数据格式说明文件。
程序中默认使用的串口名称为ttyUSB0。可以用ls命令查看实际的串口名称，并在程序中作相应修改。
有关串口的设置请参见POS320节点的说明文档。

简易使用方法：
1. 用RS232/USB转换线将XWGI5651与采集计算机连接。
2. 在采集计算机的ROS环境下启动节点：roslaunch xwgi5651 xwgi5651.launch，
   如果连线正确，节点将在新开的窗口中输出信息。
   此时测试车是静止的，组合导航系统尚未初始化，所以没有导航信息输出。
3. 开动测试车，速度大于5m/s时，组合导航系统将初始化成功，节点开始输出有效的定位数据。
